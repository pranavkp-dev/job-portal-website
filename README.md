# 💼 Job Portal Web Application – Django Project

## 📌 Overview
Job Portal is a beginner-friendly web application developed using **Python and Django** that connects job seekers and employers on a single platform. 

The system allows employers to post job openings and manage listings, while applicants can create profiles, search for jobs, and apply online. This project demonstrates core backend development concepts using Django’s MVT architecture.

---

## 🚀 Key Features

### 👤 User Module
- User Registration & Login
- Secure Authentication
- Profile Creation & Management

### 🏢 Employer Module
- Post New Jobs
- Edit & Delete Job Listings
- View Applications

### 🔎 Job Seeker Module
- Browse Available Jobs
- Search & Filter Jobs
- Apply for Jobs
- Save Job Listings

### 🛠 Admin Panel
- Manage Users
- Manage Job Posts
- Monitor Applications

---

## ⚙️ Installation & Setup

1. Clone the repository
git clone https://github.com/pranavkp-dev/job-portal-website.git

2. Navigate to project folder
cd job-portal-website

3. Install dependencies
pip install -r requirements.txt

4. Apply migrations
python manage.py migrate

5. Run server
python manage.py runserver

Open: http://127.0.0.1:8000/
